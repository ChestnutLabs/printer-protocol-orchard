*[ACE]: Anycubic Color Engine — Anycubic's multi-material feeder
*[CFS]: Creality Filament System — Creality's multi-material feeder
*[ERCF]: Enraged Rabbit Carrot Feeder — a popular DIY multi-material unit
*[mTLS]: mutual TLS — both client and server present certificates
*[TLS]: Transport Layer Security — encrypted transport
*[FTPS]: FTP over TLS — encrypted file transfer
*[MQTT]: Message Queuing Telemetry Transport — a publish/subscribe messaging protocol
*[SSDP]: Simple Service Discovery Protocol — a UPnP-style LAN discovery mechanism
*[mDNS]: multicast DNS — .local name/service discovery (Bonjour / Zeroconf)
*[UDP]: User Datagram Protocol — connectionless packets, often used for discovery
*[LAN]: Local Area Network
*[JSON-RPC]: JSON Remote Procedure Call — request/response encoded as JSON
*[REST]: Representational State Transfer — resource-oriented HTTP API style
*[SSE]: Server-Sent Events — a one-way server-to-client push stream over HTTP
*[poll-synth]: synthesizing a change-event stream by diffing successive polls of a poll-only printer
*[RFID]: Radio-Frequency IDentification — the tag some spools carry for auto-detection
